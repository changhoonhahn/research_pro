I have put together a uniform set of response functions for the
2.5-meter (henceforth SDSS), USNO, PTNew (after July 2001) and a set of
functions to cover the range of ghastly behavior of the PTOld (before
July 2001), and tables of magnitudes and colors for these response
functions run against the stars in the Gunn-Stryker spectrophotometric
atlas for SDSS, NewPT, and USNO.  These are to be found in

    ftp://astro.princeton.edu/jeg/SDSSPhotometry/ResFuns

in appropriatetely named directories. In each is an explanatory readme.txt
file, the response functions for 0, 1.0, 1.3, 1.6, 1.9 airmasses,
and in the cases mentioned, the G-S photometry files.

--jim
